Desarrollada con React JS.</p><h2>Technologies Used</h2>
<hr><ul>
<li>HTML</li>

<li>React</li>
</ul>

<hr><ul>
<li>Hooks (useState,useEffect,useParams,React-router-dom,context).</li>
